Creating new test?

  • Start date is the date after that students can attempt the test.Before this date students could not attempt the test but they can only see the test.
  • End date is the date after that students can not attempt the test.
  • You need to select the batches for whom you are making the test.
  • You can add multiple sections to your test like Physics, Chemistry, Mathematics etc.
  • After this you need to select questions for this test.Don't worry if you didn't or cannot add questions now, you can add or delete questions from your test using Test>My Tests option.
  • You can also edit start or end time of test using Test>My Tests>Edit option.
  • You can also activate or deactivate test using Test>My Tests>Activated option.Deactivated tests will not be visible to students.

Adding Questions?

  • Excel files must follow the format provided in sample file.
  • You can provide any number of options from 1-6.
  • Correct Answer must be in comma separated form of option numbers.
  • You need to specify the topic for each question so that later you can easily find them.
