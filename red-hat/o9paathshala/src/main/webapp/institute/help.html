How to use?

Step I

Create batch using Batch > Add Batch option.You can also add students to the batch using xlsx format excel file.For sample format click download.

Step II

Add students to the batches you created in Step I using Student>Add Students option. Here you can add student one by one or using excel files same as above.

Step III

This is optional, add faculty to your institute and assign batches to them using Faculty>Add Faculty option.

Step IV

Add subjects to your institute using Subjects>Add Subject option.

Step V

Create your own question bank by using Question Bank>Add Questions option. Here also either you can add question one by one or upload questions in bulk using xlsx format excel file. For sample format click download.

You can also purchase the questions from the o9 Paathshala question bank using Question Bank>Buy Questions option.

Step VI

Create a new test by using Test>New option.Then after completing the first stage you will be redirected to new stage Do not refresh this page. Here you have to add questions to each section of your test.

Step VII

View results of your students in tests by using Results option.

Adding Questions?

  • Excel files must follow the format provided in sample file.
  • You can provide any number of options from 1-6.
  • Correct Answer must be in comma separated form of option numbers.
  • You need to specify the topic for each question so that later you can easily find them.

Creating new test?

  • Start date is the date after that students can attempt the test.Before this date students could not attempt the test but they can only see the test.
  • End date is the date after that students can not attempt the test.
  • You need to select the batches for whom you are making the test.
  • You can add multiple sections to your test like Physics, Chemistry, Mathematics etc.
  • After this you need to select questions for this test.Don't worry if you didn't or cannot add questions now, you can add or delete questions from your test using Test>All option.
  • You can also edit start or end time of test using Test>All>Edit option.
  • You can also activate or deactivate test using Test>My Tests>Activated option.Deactivated tests will not be visible to students.
